Implement a public ranking system for clusters and countries that displays activity, funds, and progress in the Global Disciples process.

## Dependencies
- R-#152 (Profiles of Church / GD Cluster)
- R-#153 (Encrypted Profile Data)
- Existing SLEARN and SBT tracking system

---

## 1. Ranking Components

### 1.1 Cluster Ranking

| Column | Description | Source |
|--------|-------------|--------|
| **Position** | Rank order (1, 2, 3...) | Calculated |
| **Cluster Name** | Name set by pastors/delegates | Cluster page |
| **Country** | Country (or 🌍 for anonymous) | Cluster settings |
| **SLEARN Generated** | Total SLEARN earned by cluster members | On-chain/DB |
| **SBTs Obtained** | Total SBTs earned by cluster members | Credentials contract |
| **Milestones** | Number of GD milestones achieved | Manual tracking |
| **Activity Score** | Meetings, reports, participation | Manual tracking |
| **Funds (USDT)** | USDT in cluster fund | Contract |
| **Funds (SLEARN)** | SLEARN in cluster fund | Contract |
| **Total Score** | Weighted sum of all components | Calculated |

### 1.2 Country Ranking

| Column | Description | Source |
|--------|-------------|--------|
| **Position** | Rank order (1, 2, 3...) | Calculated |
| **Country** | Country name | Cluster settings |
| **Clusters** | Number of clusters in the country | DB |
| **Total SLEARN** | Total SLEARN generated in country | On-chain/DB |
| **Total SBTs** | Total SBTs earned in country | Credentials contract |
| **Funds (USDT)** | USDT in country fund | Contract |
| **Funds (SLEARN)** | SLEARN in country fund | Contract |
| **Total Score** | Weighted sum of all components | Calculated |

## 2. Score Calculation

### 2.1 Cluster Score Formula

| Component | Weight | Calculation |
|-----------|--------|-------------|
| SLEARN Generated | 25% | Total SLEARN earned by all cluster members |
| SBTs Obtained | 20% | Total SBTs earned by all cluster members |
| Milestones Achieved | 25% | Number of GD milestones completed |
| Activity Score | 15% | Meetings, reports, participation |
| Funds (USDT) | 10% | USDT in cluster fund |
| Funds (SLEARN) | 5% | SLEARN in cluster fund |

**Total Score** = (SLEARN × 0.25) + (SBTs × 0.20) + (Milestones × 25) + (Activity × 15) + (USDT × 10) + (SLEARN_fund × 5)

### 2.2 Country Score Formula

| Component | Weight | Calculation |
|-----------|--------|-------------|
| Clusters | 20% | Number of clusters in the country |
| Total SLEARN | 25% | Total SLEARN generated in country |
| Total SBTs | 20% | Total SBTs earned in country |
| Funds (USDT) | 20% | USDT in country fund |
| Funds (SLEARN) | 15% | SLEARN in country fund |

## 8. Database Schema

### 8.1 New Tables

```sql
-- Cluster table (extends R-#152)
CREATE TABLE cluster (
    id SERIAL PRIMARY KEY,
    name VARCHAR(100),
    country VARCHAR(2), -- NULL if anonymous
    is_anonymous BOOLEAN DEFAULT FALSE,
    multisig_address VARCHAR(42),
    verified BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- Cluster milestones
CREATE TABLE cluster_milestone (
    id SERIAL PRIMARY KEY,
    cluster_id INTEGER REFERENCES cluster(id),
    milestone_type VARCHAR(50), -- 'formation', 'course', 'director', 'trained', 'disciples', 'church', 'alliance'
    achieved_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    notes TEXT
);

-- Cluster activities
CREATE TABLE cluster_activity (
    id SERIAL PRIMARY KEY,
    cluster_id INTEGER REFERENCES cluster(id),
    activity_type VARCHAR(50), -- 'meeting', 'member_active', 'report'
    points INTEGER DEFAULT 0,
    activity_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    notes TEXT
);
```

### 8.2 Existing Table Updates

```sql
-- Add to usuario table (from R-#152)
ALTER TABLE usuario ADD COLUMN cluster_id INTEGER REFERENCES cluster(id);

-- Add to iglesia table (from R-#152)
ALTER TABLE iglesia ADD COLUMN cluster_id INTEGER REFERENCES cluster(id);

-- Add score cache table
CREATE TABLE cluster_score_cache (
    cluster_id INTEGER PRIMARY KEY REFERENCES cluster(id),
    total_score INTEGER,
    slearn_score INTEGER,
    sbt_score INTEGER,
    milestone_score INTEGER,
    activity_score INTEGER,
    fund_score INTEGER,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```
