# R-#165: Script to Create New Courses

Automate the technical steps of creating a new learn.tg course so that only content writing and Rails admin data entry remain manual.

## Dependencies
None

---

## What the Script Does

### ✅ Done

| Step | Status | Details |
|------|--------|---------|
| Guide structure | ✅ Done | Creates `resources/{en,es}/{prefix}/` and template `.md` files |
| DB record generation | ✅ Done | Generates SQL with INSERTs for `cor1440_gen_proyectofinanciero` and `cor1440_gen_actividadpf` with auto-calculated IDs |
| Vault command | ✅ Done | Prints `createVault` command with correct args (human reviews before executing on-chain) |
| Credential registration | ✅ Done | Prints `register-type` command with correct args |
| Metadata sync | ✅ Done | Prints `sync-cache` command |
| Documentation | ✅ Done | `doc/how-to-create-a-course.md` — full 7-step guide |

**Design decision:** On-chain operations (vault creation, credential registration) are printed, not auto-executed. The human reviews before running. This is intentional — no script should move funds or register credentials without human verification.

### 🔜 Pending

| Step | Status | Details |
|------|--------|---------|
| **Auto-detect course ID** | Pending | Query Rails DB for `MAX(id) + 1` instead of passing as argument |
| **Spanish course record** | Pending | Also generate `cor1440_gen_proyectofinanciero` row for `idioma='es'` with different prefix |

## Current Script

**Location:** `apps/nextjs/scripts/create-course.sh`

**Usage:**
```bash
./scripts/create-course.sh "Global Disciples" 5 "global-disciples" 4 "1.0" "1.0" "source/global-disciples.svg"
```

**What it produces:**
```
resources/
  en/global-disciples/guide1.md  → Template with 5-pillar structure
  en/global-disciples/guide2.md
  en/global-disciples/guide3.md
  en/global-disciples/guide4.md
  es/discipulos-globales/guia1.md  → Spanish template
  es/discipulos-globales/guia2.md
  es/discipulos-globales/guia3.md
  es/discipulos-globales/guia4.md
```

Plus printed commands for vault, credential, and Rails admin steps.

## Next Phase

Two improvements remaining:

1. **Auto-detect course ID:** Query `SELECT MAX(id) + 1 FROM cor1440_gen_proyectofinanciero` instead of requiring the user to pass it as argument.
2. **Bilingual course records:** Currently generates only English (`idioma='en'`). Also need a Spanish row with `idioma='es'` and the Spanish prefix.

The script already generates SQL (not just prints instructions) — this was an important upgrade from the initial version.
