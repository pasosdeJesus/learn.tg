# R-#161 Course "Tools to Bring Global Disciples to Your Cluster"

## Objective
Design and implement the pedagogical content for a new premium course that guides pastors through the Global Disciples cluster system. The course uses the existing learn.tg platform — no new Next.js features needed for the course itself. Cluster features are covered in #152.

## Dependencies

| REQ | Description | Affects |
|-----|-------------|---------|
| R-#128 | Premium course payments with dynamic pricing | Course payment |
| R-#148 | Premium payments with SLEARN & advanced features | Course payment |
| R-#152 | Church profiles + cluster system | Guide 2 (clusters must exist) |
| R-#154 | Cluster/country ranking | Displayed after Guide 2 |
| R-#156 | Donation system for clusters | Displayed after Guide 3 |
| R-#163 | Referral program | Activated upon course payment |

**Note:** Guides 1 and 2 can be written independently (content only). Guides 3 and 4 need the referenced systems to be operational.

### 2. Course Rewards

| Reward | Amount | Notes |
|--------|--------|-------|
| **USDT** | 1 × (profileScore / 100) | Paid from course vault; subject to availability |
| **SLEARN** | 5 × (profileScore / 100) | Paid from course vault; subject to availability |

**Note:** The course vault balance is public. When the balance is insufficient for a scholarship, any donation to the vault enables the next user who completes a guide to receive the scholarship. This creates a virtuous cycle: users can monitor the vault, donate to activate it, and complete a guide quickly to recover their donation and earn the rest.

### 3. Guide 1: What is Global Disciples?

**Learning objective:** Understand the mission and vision of Global Disciples and why this course matters.

**Content outline:**
- What is Global Disciples? (history, mission, global reach)
- The vision for Sierra Leone: why church clusters matter
- How the pdJ ecosystem supports the mission (learn.tg + SLEARN + SBTs)
- How the referral program works (activated upon payment)
- Why this course is premium: your payment supports the cluster system

**Reward:** 1 USDT + 5 SLEARN (subject to vault availability)

**Action:** After this guide, the pastor should understand the GD model and the course structure.

### 4. Guide 2: Form Your Cluster

**Learning objective:** Learn how to create and manage a church cluster using the platform.

**Content outline:**
- What is a cluster of churches and why form one?
- How to declare your church on learn.tg (from #152)
- How to create a cluster and get a join code
- How to invite other pastors to join your cluster
- How to join an existing cluster with a code

**Practical action:** After this guide, the pastor should create or join a cluster (when #152 is operational).

**Reward:** 1 USDT + 5 SLEARN (subject to vault availability)

### 5. Guide 3: Fund Your Cluster Mission

**Learning objective:** Understand how to raise and manage funds for the cluster.

**Content outline:**
- How donations flow through the SLEARN ecosystem
- How to set up a multisig wallet for the cluster
- How the cluster donation page works (from #156)
- Transparency: how to track donations on the platform
- How the course vault works: public balance, donations, and scholarships

**Practical action:** After this guide, the pastor should set up the cluster donation page (when #156 is operational).

**Reward:** 1 USDT + 5 SLEARN (subject to vault availability)

### 6. Guide 4: Sustain and Grow

**Learning objective:** Learn practices for long-term cluster health and growth.

**Content outline:**
- How to add more churches to the cluster
- Best practices for cluster leadership and communication
- Using the ranking system to inspire healthy growth (#154)
- Understanding the course vault cycle: donate, complete, earn
- Next steps: contacting GD, saving funds, releasing for projects

**Reward:** 1 USDT + 5 SLEARN (subject to vault availability)

**SBT:** Upon completing all 4 guides, the user receives the "Global Disciples Advocate" SBT.
