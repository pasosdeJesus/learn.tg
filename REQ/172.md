# R-#172: E2E Session Persistence Test with Puppeteer

## Objective
End-to-end test using Puppeteer to verify wallet session behavior across page
navigation on learn.tg. Tested against production (`https://learn.tg`).

## Test File

`apps/nextjs/e2e/specs/auth-session.spec.mjs`

> **Updated 2026-07-02:** Tests migrated from `test/puppeteer/` to `e2e/specs/`.
> Helpers now live in `@pasosdejesus/m/e2e`. See [R-#11](https://github.com/pasosdeJesus/m/blob/main/REQ/11.md).

## Execution

```bash
cd apps/nextjs
CHROME_PATH=/usr/local/bin/chrome IPDES=learn.tg node e2e/specs/auth-session.spec.mjs
```

Or via CLI:
```bash
cd apps/nextjs && make test-e2e-auth-session
```
