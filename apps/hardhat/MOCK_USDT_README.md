# MockUSDT Token

This is a simple mock implementation of USDT token for testing purposes on the 
Sepolia testnet.

## Deployment

The MockUSDT token has been deployed to Sepolia at:
`0x0d130F97fB5349656F95ad3Ab46BC0b34a8556a6`

The owner (deployer) has 1,000,000 tokens with 6 decimals.

## Available Scripts

- `yarn deploy-mock-usdt` - Deploy a new MockUSDT token to Sepolia Celo
- `yarn verify-mock-usdt` - Verify the deployed MockUSDT token

## Token Details

- Name: Mock USDT
- Symbol: MUSDT
- Decimals: 6
